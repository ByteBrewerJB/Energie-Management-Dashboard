# Energie-Management Dashboard

Deze repository bevat de documentatie en broncode voor de energie-management webapplicatie. De applicatie combineert energieverbruik, opwek, tarieven, CO₂-impact, investeringsanalyses en automatisering voor huishoudens en MKB.

## Documentatie

* [Totale specificatie en self-hosted instructies](docs/specification.md)

De specificatie bevat zowel de functionele en technische eisen als de volledige Docker/Portainer-instructies voor self-hosted deployments in 2025.

## Status

De repository bevat momenteel de documentatie die als basis dient voor de implementatie. Verdere code en configuraties worden in toekomstige commits toegevoegd.
